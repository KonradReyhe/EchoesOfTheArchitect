# Echoes of the Architect

A spiritual journey game built with Python and Pygame.

## Description
Explore a mystical garden where every choice matters. Make meaningful decisions that shape your path through this allegorical experience.

## Features
- Atmospheric garden scene with interactive elements
- Meaningful choice system (limited to 2 major decisions)
- Beautiful artwork and symbolic imagery
- Start screen with title and prompt

## Requirements
- Python 3.x
- Pygame
